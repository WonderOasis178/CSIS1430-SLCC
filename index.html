Reflection

From this class, I learned HTML, CSS, JavaScript, a little bit of Bootstrap, and how to apply those things to make a website. There was some challenge from this class to learn the materials, mostly JavaScript, but it was fun to learn all those things to create a website and make it aesthetically pleasing. The most interesting thing I have learned is the advanced CSS to apply to a webpage, making it look nice. I loved having a hard time learning about Bootstrap and how to apply it while at the same time being able to customize how the website looks. I enjoyed this class very much, and thanks to professor Stone teaching us about the subject matter in this class, even if his videos were way too long. I think this class created a good basis for me to work on in the future, and will definitely continue to work on them and revisit my projects.
